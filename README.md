## ğŸ”§ Installation

1. Clone repo

   ```bash
   git clone https://github.com/Nizar4code/nizar4code.github.io.git
   ```

2. Install dependencies

   ```bash
   pnpm install
   # or
   npm install
   ```

   npm run dev
